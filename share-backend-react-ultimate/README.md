Các bước cần làm để chạy dự án Backend: (đã cài đặt môi trường Node.JS)

1. Cài đặt thư viện cần thiết
- Sử dụng npm:   npm install
- Sử dụng yarn:  yarn

2. Sửa đổi file .env (nếu cần thiết)

3. Chạy dự án backend :  npm start
